# Overview

This project demonstrates a complete data analytics workflow starting from data collection and preprocessing to dashboard creation and reporting. The objective of the project is to extract meaningful insights from raw data using analytical and visualization tools.

The project includes:

* Loading and processing datasets in Python
* Performing Exploratory Data Analysis (EDA)
* Cleaning and transforming data
* Running SQL queries using PostgreSQL
* Building an interactive Power BI dashboard
* Creating analytical reports
* Preparing a presentation using Gamma

This project showcases practical data analytics skills commonly used in real-world business environments.

# How to Run the Project

## 1. Clone the Repository

```bash
git clone <repository-link>
```

## 2. Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn psycopg2
```

## 3. Run Jupyter Notebook

```bash
jupyter notebook
```

## 4. PostgreSQL Setup

* Create a PostgreSQL database
* Import cleaned dataset
* Run SQL scripts from the `sql_queries/` folder

## 5. Open Power BI Dashboard

* Open `.pbix` file in Power BI Desktop
* Refresh dataset connection if needed
